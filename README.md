# Coming Soon

The Coming Soon module allows you to display a "coming soon" page, with an
optional countdown timer and customizable appearance, while keeping normal
web access to users with proper permissions or to certains paths (admin paths
and user/login at least).

## Installation

- Install this module using [the official Backdrop CMS instructions](https://docs.backdropcms.org/documentation/extend-with-modules).

## Configuration

After enabling the module, configure it at `Administration » Configuration »
System » Coming Soon` (`admin/config/system/coming-soon`).
